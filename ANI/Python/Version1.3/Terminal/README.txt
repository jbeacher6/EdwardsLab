Anilyze Version 1.0

Prerequisite Instructions: 
1. Download all fna files from ftp://ftp.ncbi.nlm.nih.gov/genomes/archive/old_refseq/Bacteria/all.fna.tar.gz to an empty directory 
2. Unzip and delete the tar.gz file
Prerequisite Software: Python 2.7, Mac OSX or a Linux Operating System
Final results will be found in outDirectoryGiven/complete/complete/

To Execute After Prerequisites Completed: 
python anilyzePrep.py -i <unzipped fna files directory(all.fna.tar.gz) location> -o <forward and reverse compliment fna files directory location>
